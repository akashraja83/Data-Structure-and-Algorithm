# Doubly Linked List Implementation (Java)

    This project contains a complete implementation of a Doubly Linked List in Java.
    Each node contains:
    - data
    - next pointer
    - previous pointer


Features Implemented

    Insertion
    - Insert at Beginning
    - Insert at End
    - Insert at Specific Position (0-based)
    
    Deletion
    - Delete from Beginning
    - Delete from End
    - Delete at Specific Position
    
    Utility
    - Count Nodes
    - Search for a Value
    - Find Middle (Slow & Fast Pointer)
    - Display Forward
    - Display Backward
