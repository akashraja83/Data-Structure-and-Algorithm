Circular Doubly Linked List Implementation (Java)

    -This project contains a complete implementation of a Circular Doubly Linked List in Java.
    -In a Circular Doubly Linked List, each node contains a reference to both the next node and the previous node, and the last node points back to the head node, forming a circular chain.
    -Unlike a normal Doubly Linked List, traversal never reaches null. You can traverse forward or backward until you reach the head node again.
    -Each node contains:
      -data
      -next pointer
      -prev pointer
      
Features Implemented

    Insertion
      -Insert at Beginning
      -Insert at End
      -Insert at Specific Position (0-based)
      
    Deletion
      -Delete from Beginning
      -Delete from End
      -Delete at Specific Position
    
    Utility Methods
      -Count Nodes – Returns the total number of nodes in the list
      -Search for a Value – Checks if a value exists in the list
      -Display Elements – Traverses the list forward and prints all elements
