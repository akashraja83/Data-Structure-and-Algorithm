# Circular Linked List Implementation (Java)

    This project contains a complete implementation of a Circular Linked List in Java
    using the Singly Linked List structure.

    In a Circular Linked List, the last node points back to the head node
    instead of pointing to null. This forms a circular chain of nodes.

    Each node contains:
    - data
    - next pointer

    Unlike a normal Singly Linked List, traversal never reaches null.
    The list continues until it reaches the head node again.


Features Implemented

    Insertion
    - Insert at Beginning
    - Insert at End
    - Insert at Specific Position (0-based)

    Deletion
    - Delete from Beginning
    - Delete from End
    - Delete at Specific Position


    - Count Nodes
    - Search for a Value
    - Display Elements
